cfi30's dotfiles
================

Installation (asks before overwriting any file):
```
$ git clone https://cfi30@bitbucket.org/cfi30/dotfiles.git ~/cfi30-dotfiles
$ cd ~/cfi30-dotfiles
$ ./install.sh */ # full installation
$ ./install.sh i3 xorg zsh ... # partial installation
```
